Windows operációs rendszerek története

Forrás: The History of Microsoft Operating Systems

MS-DOS (Microsoft disk operating system)

A Microsoft fejlesztette ki az IBM számára. Sokáig standard operációs rendszernek számított az IBM kompatibilis rendszerekhez. Kezdetekben nagyon hasonlított egy korábbi operációs rendszerre a CP/M-re.

Windows 1.0 – 3.1 (1985-1992)

DOS parancsok helyett rámutatás és kattintás a funkciók eléréséhez. Ablak fogalom bevezetése. Az asztal és az ikonok a 2.0-ás verziótól érhetőek el.

A 3.0-ás verzióban elérhető a fájl- és nyomtató kezelő valamint a különböző játékok.

Windows 95, 98, ME

DOS alaprendszertűl független verzió. Ez lehetővé tette sok DOS alapú limitáció megszegését. Először kerül támogatásra a 32bit-es rendszer és a plug-n-play. A Windows 98 több új technológia támogatását adta hozzá és bevezette az Active Desktop-ot (böngésző és operációs rendszer integráció).

Windows NT: Windows NT Server, Windows NT Workstation

NT 3.1 - 4.0 (1993-1996): New Technology, 32bit operációs rendszer család, szerverek és munkaállomások számára.

Windows 2000 (February 2000): üzleti felhasználókat célozza. Létezett belőlle szerver és munkaállomás verzió is.

Windows Server 2008, ... párhuzamos fejlődése és kiadása.

Új grafikus irányok

Windows XP, 7, Vista újratervezett grafikus felület folyamatosan integrálva az új technológiákat.

Mobil platformok

Windows CE, Windows Mobile, Windows Phone (Nokia)

Windows 8: teljesen új desing, csempék.

Windows 10: start menü kálvária.
